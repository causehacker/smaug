Official Chrome browser extension that brings Claude's AI assistance directly into your browser. Works seamlessly with Claude Code and Desktop to enable browser automation, data analysis, and site navigation through natural language commands.

## Key Features

- Ask questions and get AI assistance while browsing
- Analyze data and automate tasks on web pages
- Navigate and interact with websites through Claude
- Integration with Claude Code for browser automation (`claude --chrome`)
- Compatible with Claude Desktop app

## Use Cases

- Automate repetitive web tasks
- Extract and analyze data from websites
- Modify page styling and elements via natural language
- Research and information gathering across multiple sites
- Web scraping and data collection
