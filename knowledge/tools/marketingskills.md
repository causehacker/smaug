A collection of AI agent skills focused on marketing tasks for Claude Code. Built for technical marketers and founders who want Claude Code to help with conversion optimization, copywriting, SEO, analytics, and growth engineering.

## Key Features

- 21 specialized marketing skills covering:
  - Conversion Rate Optimization (CRO): page-cro, form-cro, signup-flow-cro, onboarding-cro, paywall-upgrade-cro, popup-cro
  - Copywriting: copywriting, copy-editing, email-sequence
  - SEO: seo-audit, programmatic-seo, schema-markup
  - Analytics & Testing: ab-test-setup, analytics-tracking
  - Growth: launch-strategy, referral-program, free-tool-strategy, paid-ads
  - Strategy: pricing-strategy, competitor-alternatives, marketing-ideas (140 SaaS ideas), marketing-psychology (70+ mental models)
  - Content: social-content

## Installation

Via add-skill CLI (Recommended):
```bash
# Install all skills
npx add-skill coreyhaines31/marketingskills

# Install specific skills
npx add-skill coreyhaines31/marketingskills --skill page-cro copywriting
```

## Usage

Skills auto-trigger based on keywords (e.g., "A/B test", "optimize page", "write copy") or invoke directly in Claude Code.
