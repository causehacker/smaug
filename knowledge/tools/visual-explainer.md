Agent skill + prompt templates that generate rich HTML pages for visual diff reviews, architecture overviews, plan audits, data tables, and project recaps. The Visual Explainer reduces cognitive debt by transforming complex terminal-based information into visually organized, well-designed HTML pages. It includes reference templates and a CSS pattern library to ensure consistent design quality.

## Key Features

- Agent skill for use with Claude Code and similar agent frameworks
- Generates rich HTML pages for various use cases:
  - Visual diff reviews
  - Architecture overviews
  - Plan audits
  - Data tables
  - Project recaps
- Reference templates included
- CSS pattern library for consistent design
- Simplifies complex information presentation
