A stop hook for Claude Code that keeps the agent working until all plans and user requests are 100% complete. This tool prevents Claude Code agents from stopping prematurely, ensuring continuous execution until tasks are fully finished.

## Key Features

- Stop hook integration with Claude Code
- Keeps agents working until completion
- Plans and request fulfillment tracking
- Automation for extended agent runs
