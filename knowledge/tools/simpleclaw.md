SimpleClaw is a one-click deployment tool for OpenClaw that removes all technical complexity. It allows users to deploy their own 24/7 active OpenClaw instance in under 1 minute, making the powerful AI agent platform accessible to non-technical users.

## Key Features

- **Ultra-fast deployment**: Deploy OpenClaw in under 1 minute
- **One-click setup**: No technical configuration required
- **24/7 availability**: Keeps OpenClaw instance running continuously
- **Removes barriers**: Makes OpenClaw accessible to "all normal people" without DevOps knowledge
- **Weekend project**: Built as a rapid side project to simplify OpenClaw adoption

## What It Solves

OpenClaw typically requires technical setup including server configuration, API keys, and ongoing maintenance. SimpleClaw abstracts all of this away, allowing anyone to deploy a fully functional OpenClaw instance without understanding the underlying infrastructure.
